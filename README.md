Pimatic plugin for Nuki Smart Locks

## Plugin Configuration

    {
          "plugin": "nuki",
          "active": true,
          "debug": false,
          "host": "localhost",
          "port": 8881,
          "token": "changeme"
    }

## Device Configuration

Use the "Discover Devices" function provided by pimatic to automatically discover and setup devices.
